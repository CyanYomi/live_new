# 九里羊汤馆（园林店）

#111

## :mag: 位置

高德和大众点评上都可以找到，在鼓楼区平山路平山寺那边，离学校有些远，建议几个人一块去吃，地方比较偏僻，但是店里环境还行，挺大的，还有个小院子。

## 🌰 食评

当时来学校这边上学吃的第一顿就是羊肉，虽然说徐州特色是狗肉，但是羊肉也是不错的，所以一直想再吃徐州的羊肉，这次就特地找了羊肉馆子。

【点单】：肚包肉、羊蝎子、羊肉串和凉拌羊肚。

【肚包肉】：个人认为是这次点的最好吃的菜。羊肚里面裹的羊肉非常嫩！！！！羊肚有韧劲儿，但是不会嚼不动，羊肉处理的很好一点羊膻味没有，羊肚跟里面的羊肉就着酱料一吃真的很满足。我肯定会为了这口肚包肉再来的。

【羊肉串】：进店的时候可以看到是放在炭火上现烤的，很好吃，调味也恰到好处，不油，火候把握的刚刚好。

【羊蝎子】：感觉调味不太够，有些清淡了，调味一清淡之后就只剩辣味，汤汁如果再浓郁一点会更好吃，不过听说羊蝎子的调味有时候足有时候不太足，感觉菜品的出品不太稳定，同样没有什么膻味，适合喜欢吃肉骨架的，骨头里面的髓质可以一起吃掉。

【凉拌羊肚】：可能是天冷了，店家给加热了变成了温拌羊肚，醋一加热就发酸，导致整盘羊肚都有点酸，调味上扣了不少分，但是肉不错。
